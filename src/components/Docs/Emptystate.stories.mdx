# Empty states

Prvek který zobrazuje, že nejsou k dispzici žádné data, například když vyhledávání nemá žádné výsledky.

## Anatomie prvku

Prázdné stavy se zobrazují v různých scénářích.

1.  Obsah stránky se nemůže načíst.
2.  Žádné výsledky pro vyhledávání, filtrace
3.  Prázdné výsledky další funkce v systému.
    

Uživatelé někdy provedou akci a očekávají, že něco uvidí, ale nemáte co jim ukázat. K tomu dochází, když vyhledávání nemá žádné výsledky. Pomozte jim dostat se z této situace poskytnutím pomoci.
