# Inline dialog

Inline dialogy je vyskakovací okno, které slouží k zobrazení menšího množství informací nebo akčních prvků, nejsou tak rušivým elementem jako Modální dialog.

## Varianty
<img src="https://via.placeholder.com/468x400?text=desktop + mobile" />

1.  Trigger dialogu:
    
2.  Obsah dialogu
    
3.  Ovládací prvky: Ovládací prvky mohou zahrnovat akční prkvy (tlačítka, odkazy, checkboxy)
    

## Kdy a jak používat

- Vložené dialogy by měly být použity, pokud oddíly na stránce vyžadují další informace nebo akce, ale nejsou pro stránku jako celek zásadní.

-  V ideálním případě by měly být inline dialogy stručné, pro efektivní dosažení cíle.
    
-  Zobrazujte pouze jeden dialog na stránce.
    
-  Častým použitím pro inline dialogy je např. vyhledávání. (youtrack, hledání a přidávání hodnot)

## Chování

Inline dialog se zobrazí, po interakci uživatele (kliknutím na tlačítko)

### Scroll

U dlouhých seznamů v inline dialogu se obsah posouvá, zatímco záhlaví zůstává pevné. Pokud potřebujete zobrazit mnoho obsahových nebo složitých interakcí, zvažte místo toho [modální dialog](https://atlassian.design/guidelines/product/components/modal-dialog) .
