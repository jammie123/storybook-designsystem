# Badge
Prvek sloužící prezentování krátkých relevantních informací.

Pokud máte komplexní uživatelské rozhraní, uživatelé se někdy mohou snažit najít relevantní informace. Pomocí kompoennty badge upozorněte na informace způsobem, který usnadňuje skenování.
Badge by měly uvádět pouze informace, například stavy, ale neměly by mít žádné interakce.
