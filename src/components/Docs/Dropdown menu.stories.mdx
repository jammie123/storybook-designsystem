# Dropdown menu

Rozbalovací nabídka s možnostmi.

  

## Anatomie prvku
<img src="https://via.placeholder.com/468x400?text=Ukázka dropdown menu popis částí" />

## Varianty
<img src="https://via.placeholder.com/468x400?text=Desktop a mobile" />

1.  Tlačítko - triger pro otevření menu, může být tlačítko nebo i jiný akční prvek.
    
2.  Trigger
    
3.  Obsah menu - může obsahovat text i akční prvky (odkazy + tlačítka)
    

  

## Kdy a jak používat 

- U dlouhých seznamů seskupte související položky.
- Obsahem menu, může být i radibutton nebo checknox.
- Rozbalovací nabídky se obvykle používají, když máte na výběr 5-15 položek.
- V případech, kdy je položka nabídky delší než textový popisek tlačítka, se nabídka zvětší na šířku nejdelší uvedené položky. Komponenta však určuje maximální šířku. Pokud položka nabídky přesáhne maximální šířku, bude zkrácena.
